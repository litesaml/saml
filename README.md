# About Lite Saml

[![Run tests](https://github.com/litesaml/saml/actions/workflows/run_tests.yml/badge.svg)](https://github.com/litesaml/saml/actions/workflows/run_tests.yml)
[![Run quality checks](https://github.com/litesaml/saml/actions/workflows/run_quality_checks.yml/badge.svg)](https://github.com/litesaml/saml/actions/workflows/run_quality_checks.yml)
[![Latest Stable Version](https://poser.pugx.org/litesaml/saml/v/stable)](https://packagist.org/packages/litesaml/saml)

High-level SAML 2.0 PHP library built on top of [litesaml/lightsaml](https://github.com/litesaml/lightsaml).

Provides `ServiceProviderWrapper` and `IdentityProviderWrapper` to handle common SAML flows — authentication requests and responses, logout, and signature validation — using PSR-7 HTTP interfaces.

## Installation

```bash
composer require litesaml/saml
```
